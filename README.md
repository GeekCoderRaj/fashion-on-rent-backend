## Instructions to run


* Directions to setup/install
```bash
git clone https://github.com/GeekCoderRaj/fashion-on-rent-backend.git
npm install
Add APP_PORT, MONGO_URL, SECRET, REFRESH_SECRET, CLOUD Confiuration in .env
```

* Directions to execute

```bash
npm run dev
```
